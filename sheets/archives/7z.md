# 7z (7-Zip archiver)

High-compression archiver supporting multiple formats with AES-256 encryption.

## Create Archives

### Add Files

```bash
# Create a 7z archive
7z a archive.7z file1.txt file2.txt

# Archive a directory
7z a archive.7z /path/to/directory/

# Create zip format instead of 7z
7z a -tzip archive.zip /path/to/directory/

# Create tar.gz (gzip tar)
7z a -ttar archive.tar /path/to/directory/
# Then: 7z a -tgzip archive.tar.gz archive.tar
```

### Compression Level

```bash
# Store only (no compression)
7z a -mx0 archive.7z /var/data/

# Fastest
7z a -mx1 archive.7z /var/data/

# Normal (default)
7z a -mx5 archive.7z /var/data/

# Maximum
7z a -mx7 archive.7z /var/data/

# Ultra (slowest, best ratio)
7z a -mx9 archive.7z /var/data/

# Ultra with LZMA2 (best for large files)
7z a -mx9 -m0=lzma2 archive.7z /var/data/
```

## Extract Archives

### Unpack

```bash
# Extract preserving directory structure
7z x archive.7z

# Extract to a specific directory
7z x archive.7z -o/opt/output/

# Extract flat (no directory structure)
7z e archive.7z

# Extract specific files
7z x archive.7z "path/to/file.txt"

# Overwrite all without prompting
7z x -aoa archive.7z

# Skip existing files
7z x -aos archive.7z
```

## List Contents

### View Archive

```bash
# List files
7z l archive.7z

# Technical info (compression method, sizes)
7z l -slt archive.7z
```

## Password Protection

### AES-256 Encryption

```bash
# Create encrypted archive
7z a -p archive.7z /var/secrets/

# Create with password on command line
7z a -pMySecretPassword archive.7z /var/secrets/

# Encrypt filenames too (7z format only)
7z a -p -mhe=on archive.7z /var/secrets/

# Extract encrypted archive
7z x -pMySecretPassword archive.7z
```

## Exclude Files

### Exclude Patterns

```bash
# Exclude by wildcard
7z a archive.7z /path/to/project/ -x!*.log -x!*.tmp

# Exclude directories
7z a archive.7z /path/to/project/ -xr!.git -xr!node_modules

# Exclude from a list file
7z a archive.7z /path/to/project/ -x@exclude.lst

# Recursive exclude (r flag after x)
7z a archive.7z project/ -xr!__pycache__
```

## Split Archives

### Multi-Volume

```bash
# Create 100MB volumes
7z a -v100m archive.7z /var/data/

# Result: archive.7z.001, archive.7z.002, ...

# Extract from first volume (finds the rest automatically)
7z x archive.7z.001
```

## Test Integrity

### Verify

```bash
7z t archive.7z

# Test with password
7z t -pMyPassword archive.7z
```

## Update Archives

### Modify Existing

```bash
# Update (add new, replace changed)
7z u archive.7z newfile.txt

# Delete from archive
7z d archive.7z "unwanted/file.txt"
```

## Tips

- `7z x` preserves directory structure; `7z e` extracts everything flat into one directory. Use `x` almost always.
- `-mhe=on` encrypts filenames (only in 7z format, not zip). Without it, the content is encrypted but filenames are visible.
- 7z uses AES-256 encryption which is far stronger than zip's ZipCrypto. Use 7z for any sensitive data.
- The `-o` flag has no space before the path: `-o/opt/output/`, not `-o /opt/output/`.
- 7z can extract many formats it cannot create: RAR, ISO, DMG, VHD, and more.
- `p7zip` is the Linux port. Install with `apt install p7zip-full` (for the `7z` command) or `p7zip` (for the `7za` subset).
- For scripting, use `7z` return codes: 0=success, 1=warning, 2=fatal error, 7=bad command line, 8=out of memory.
